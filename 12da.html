96-<ruby>

97-<s>

La <s> etiqueta especifica texto que ya no es correcto, preciso o relevante. El texto se mostrará con una línea atravesado.

98-<samp>

La <samp> etiqueta se utiliza para definir la salida de muestra de un programa de computadora. El contenido interior se muestra en la fuente monoespaciada predeterminada del navegador.

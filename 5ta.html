Etiquetas y sus Definiciones y Uso en HTML5

30-<br>

Para indicarle al navegador que queremos que continúe en la próxima línea debemos hacerlo con el elemento HTML.

Ver Ejemplo

31-<ol>

Este etiqueta HTML es útil cuando debemos numerar o listar una serie de datos.

Ver Ejemplo

32-<ul>

Una lista no ordenada como su nombre lo indica no utiliza un número delante de cada items sino un pequeño símbolo gráfico.

Ver Ejemplo

33-<dl>

Como su nombre lo indica se utiliza para asociar un término y la definición del mismo. El navegador se encarga de destacar y separa el término y su definición.

Ver Ejemplo

34-<!-- -->

Un comentario es un texto que solo le interesa a la persona que desarrolló la página, el navegador ignora todo el contenido que se encuentra dentro de esta marca.

Ver Ejemplo

35-<form>

Un formulario permite que el visitante al sitio cargue datos y sean enviados al servidor.

Ver Ejemplo

36-<pre>

nos permite mostrar toda la información que está contenida entre las marcas de comienzo y fin tal cual como lo hemos escrito respetando inclusive los espacios en blanco y salto de líneas.

Ver Ejemplo

37-<abbr>

La <abbr>etiqueta define una abreviatura o un acrónimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Consejo: utilice el atributo global para mostrar la descripción de la abreviatura/acrónimo cuando pase el mouse sobre el elemento.

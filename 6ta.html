Etiquetas y sus Definiciones y Uso en HTML5

38-<acronym>

La <acronym> etiqueta se utilizó en HTML 4 para definir un acrónimo.

Ver Ejemplo

39-<applet>

La <applet> etiqueta se utilizó en HTML 4 para definir un subprograma incrustado (complemento).

Ver Ejemplo

40-<area>

La <area>etiqueta define un área dentro de un mapa de imagen (un mapa de imagen es una imagen con áreas en las que se puede hacer clic).

Ver Ejemplo

41-<wbr>

La <wbr> etiqueta (Oportunidad de salto de palabra) especifica en qué parte del texto estaría bien agregar un salto de línea.

Ver Ejemplo

42-<audio>

La <audio> etiqueta se utiliza para incrustar contenido sonoro en un documento, como música u otras transmisiones de audio.

Ver Ejemplo

43-<base>

La <base> etiqueta especifica la URL base y/o el destino de todas las URL relativas de un documento.

Ver Ejemplo

44-<basefont>

La <basefont> etiqueta se usó en HTML 4 para especificar un color de texto, tamaño de fuente o familia de fuentes predeterminados para todo el texto en un documento HTML.

Ver Ejemplo

45-<bdi>

La <bdi> etiqueta aísla una parte del texto que podría tener formato en una dirección diferente del resto del texto externo.

Ver Ejemplo

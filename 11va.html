Etiquetas y sus Definiciones y Uso en HTML5

80-<map>

La <map > etiqueta se utiliza para definir un mapa de imágenes. Un mapa de imágenes es una imagen con áreas en las que se puede hacer clic.

Ver Ejemplo

81-<menu>

La <menu> etiqueta define una lista desordenada de contenido.

Ver Ejemplo

82-<meter>

La <meter> etiqueta define una medida escalar dentro de un rango conocido o un valor fraccionario. Esto también se conoce como calibre.

Ver Ejemplo

83-<noframes>

La <noframes> etiqueta se utilizó en HTML 4 para actuar como etiqueta alternativa para los navegadores que no admitían marcos.

Ver Ejemplo

84-<noscript>

La <noscript> etiqueta define un contenido alternativo que se mostrará a los usuarios que tienen scripts deshabilitados en su navegador o tienen un navegador que no admite scripts.

Ver Ejemplo

85-<object>

La <object> etiqueta define un contenedor para un recurso externo.

Ver Ejemplo

86-<optgroup>

La <optgroup> etiqueta se utiliza para agrupar opciones relacionadas en un elemento <select> (lista desplegable).

Ver Ejemplo

87-<option>

La <option> etiqueta define una opción en una lista de selección.

Ver Ejemplo

88-<output>

La <output> etiqueta se utiliza para representar el resultado de un cálculo (como uno realizado por un script).

Ver Ejemplo

89-<param>

La <param> etiqueta se utiliza para definir parámetros para un elemento

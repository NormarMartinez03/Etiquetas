Etiquetas y sus Definiciones y Uso en HTML5

110-<summary>

La <summary> etiqueta define un encabezado visible para el elemento <detalles> . Se puede hacer clic en el encabezado para ver/ocultar los detalles.

ver Ejemplo

111-<sup>

La <sup> etiqueta define texto en superíndice. El texto en superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. Se puede utilizar texto en superíndice para notas a pie de página, como WWW [1].

Ver Ejemplo

112-<svg>

La <svg> etiqueta define un contenedor para gráficos SVG.

Ver Ejemplo

113-<template>

La <template> etiqueta se utiliza como contenedor para mantener contenido HTML oculto al usuario cuando se carga la página.

Ver Ejemplo

114-<tbody>

La <tbody> etiqueta se utiliza para agrupar el contenido del cuerpo en una tabla HTML.

Ver Ejemplo

115-<td>

La <td> etiqueta define una celda de datos estándar en una tabla HTML.

Ver Ejemplo

116-<textarea>

La <textarea> etiqueta define un control de entrada de texto de varias líneas.

Ver Ejemplo

117-<tfoot>

La <tfoot> etiqueta se utiliza para agrupar el contenido del pie de página en una tabla HTML.

Ver Ejemplo

118-<track>

La <track> etiqueta especifica pistas de texto para elementos <audio> o <video>.

119-<video>

La <video> etiqueta se utiliza para incrustar contenido de vídeo en un documento, como un clip de película u otras secuencias de vídeo.

Ver Ejemplo

120-<var>

La <var> etiqueta se utiliza para definir una variable en programación o en una expresión matemática. El contenido interior normalmente se muestra en cursiva .

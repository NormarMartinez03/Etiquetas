Etiquetas y sus Definiciones y Uso en HTML5

100-<search>

La <search> etiqueta se utiliza para especificar que aquí viene un conjunto de elementos relacionados con la búsqueda.

Ver Ejemplo

101-<section>

La <section> etiqueta define una sección en un documento.

Ver Ejemplo

102-<select>

El <select> elemento se utiliza para crear una lista desplegable.

Ver Ejemplo

103-<small>

La <small> etiqueta define texto más pequeño (como derechos de autor y otros comentarios laterales).

Ver Ejemplo

104-<source>

La <source> etiqueta se utiliza para especificar múltiples recursos multimedia para elementos multimedia, como <video> , <audio> y <picture> .

Ver Ejemplo

105-<span>

La <span> etiqueta es un contenedor en línea que se utiliza para marcar una parte de un texto o una parte de un documento.

Ver Ejemplo

106-<strike>

La <strike> etiqueta se utilizó en HTML 4 para definir texto tachado.

Ver Ejemplo

107-<strong>

La <strong>etiqueta se utiliza para definir texto con gran importancia. El contenido del interior normalmente se muestra en negrita.

Ver Ejemplo

108-<style>

La <style> etiqueta se utiliza para definir información de estilo (CSS) para un documento.

Ver Ejemplo

109-<sub>

La <sub> etiqueta define el texto del subíndice. El texto del subíndice aparece medio carácter debajo de la línea normal y, a veces, se representa en una fuente más pequeña. El texto con subíndice se puede utilizar para fórmulas químicas, como H 2 O.

Etiquetas y sus Definiciones y Uso en HTML5

1-<!DOCTYPE html>

Define el documento como HTML5.

Ver Ejemplo

2-<html>

Es el elemento raíz de un documento HTML.

Ver Ejemplo

3-<head>

Contiene la información del documento, como el título, los estilos CSS y las referencias a scripts JavaScript.

Ver Ejemplo

4-<title>

Agrega un título al documento. Se muestra en la pestaña del navegador o en las aplicaciones que lo utilizan.

Ver Ejemplo

5-<body>

Contiene el contenido visible de la página web, como texto, imágenes, vídeos, etc.

Ver Ejemplo

6-<h1 a h6>

Son etiquetas de encabezado que se utilizan para dar importancia a diferentes partes del contenido. h1 es el encabezado más importante, mientras que h6 es el menos importante.

Ver Ejemplo

7-<p>

Esta etiqueta se utiliza para agregar párrafos de texto al documento.

Ver Ejemplo

8-<a>

Esta etiqueta se utiliza para crear enlaces a otras páginas web o recursos.

Ver Ejemplo

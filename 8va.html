Etiquetas y sus Definiciones y Uso en HTML5

54-<colgroup>

La <colgroup> etiqueta especifica un grupo de una o más columnas en una tabla para formatear.

Ver Ejemplo

55-<data>

La <data> etiqueta se utiliza para agregar una traducción legible por máquina de un contenido determinado.

Ver Ejemplo

56-<datalist>

La <datalist> etiqueta especifica una lista de opciones predefinidas para un elemento <input>.

Ver Ejemplo

57-<dd>

La <dd> etiqueta se utiliza para describir un término/nombre en una lista de descripción.

Ver Ejemplo

58-<del>

La <del> etiqueta define el texto que se ha eliminado de un documento. Los navegadores suelen marcar una línea en el texto eliminado.

Ver Ejemplo

59-<details>

La <details> etiqueta especifica detalles adicionales que el usuario puede abrir y cerrar según demanda.

Ver Ejemplo

60-<dfn>

La <dfn> etiqueta representa el "elemento de definición" y especifica un término que se definirá dentro del contenido.

Ver Ejemplo

61-<dialog>

La <dialog> etiqueta define un cuadro de diálogo o una subventana.

Etiquetas y sus Definiciones y Uso en HTML5

62-<dir>

La <dir> etiqueta se usó en HTML 4 para enumerar títulos de directorios.

Ver Ejemplo

63-<dl>

La <dl> etiqueta define una lista de descripción.

Ver Ejemplo

64-<dt>

La <dt> etiqueta define un término/nombre en una lista de descripción.

Ver Ejemplo

65-<embed>

La <embed>etiqueta define un contenedor para un recurso externo, como una página web, una imagen, un reproductor multimedia o una aplicación complementaria.

Ver Ejemplo

66-<fieldset>

La <fieldset>etiqueta se utiliza para agrupar elementos relacionados en un formulario.

Ver Ejemplo

67-<font>

La <font> etiqueta se utilizó en HTML 4 para especificar el tipo de fuente, el tamaño de fuente y el color del texto.

Ver Ejemplo

68-<frame>

La <frame>etiqueta se usó en HTML 4 para definir una ventana (marco) particular dentro de un <frameset>.

Ver Ejemplo

69-<frameset>

La <frameset> etiqueta se utilizó en HTML 4 para definir un conjunto de marcos.

Ver Ejemplo

70-<hr>

La <hr> etiqueta define una ruptura temática en una página HTML (por ejemplo, un cambio de tema).
